# MiniDocker

## Overview

MiniDocker is a production-focused learning path designed for experienced backend engineers who want to understand Docker from first principles and use it effectively in real-world cloud-native systems.

## Expected Outcome

After completing MiniDocker, you should understand:

- Why containers exist
- Linux namespaces and cgroups
- Docker image layering
- Container networking
- Persistent storage
- Spring Boot containerization
- Docker Compose for microservices
- Production debugging
- Container security

This prepares you for MiniKubernetes and modern cloud-native architecture.
